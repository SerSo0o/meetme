# close-phase

Description: Close a Meet Me implementation phase cleanly.

1. Summarize what was completed in this phase.
2. Verify the acceptance criteria one by one.
3. List any placeholders still left.
4. List risks or edge cases not yet handled.
5. Update:
   - docs/roadmap.md
   - docs/architecture.md if structure changed
   - docs/database.md if schema changed
6. Propose the next phase in one short paragraph.
7. End with a manual smoke-test checklist.
