# Mobile UI rule

- Optimize for mobile-first readability and thumb-friendly actions.
- Prefer simple bottom navigation and predictable screen hierarchy.
- Keep cards scannable and visually calm.
- Use strong spacing and limited visual noise.
- Always include loading, empty, and error states.
- Avoid visually coding the app as a cheap hookup product.
- Keep the tone premium, safe, and social.
